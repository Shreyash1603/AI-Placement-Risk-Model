# AI Placement Risk Model

AI-powered placement risk prediction system for education loan borrowers.

## Features
- Placement prediction (3/6/12 months)
- Salary prediction
- Risk scoring
- Streamlit dashboard
- XGBoost ML model

## Tech Stack
- Python
- Pandas
- XGBoost
- Scikit-learn
- Streamlit

## Run
pip install -r requirements.txt
streamlit run app/app.py
